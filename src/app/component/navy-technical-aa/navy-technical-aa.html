Navy Senior Secondary Recruit (SSR)
1. Selection Process for Navy ( Technical SSR )
The Selection Process for Navy ( Technical SSR ) are :-
  1. Written Examination (CBT)
  2. Physical Test
  3. Medical Test
2. Educational Qualification for Navy ( Technical SSR )
12th Pass with Maths/Physics/Chemistry.
3. Age Limit for Navy ( Technical SSR )
The Age limit for Navy ( Technical SSR ) are
  1. Minimum - 17.5 Years
  2. Maximum - 21 Years
4. Physical Measurement Test​ for Navy ( Technical SSR )
In Navy ( Technical SSR ) physical test, the eligibility of candidates is measured as per height, chest and medical standards.
  • Height – 157
  • Weight – Weight should be proportionate to height
  • Chest – Minimum Expansion of 5 cms
