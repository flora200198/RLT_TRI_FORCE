Army ( Technical )
1. Selection Process for Army ( Technical )
The Selection Process for Army ( Technical ) are :-
  1. Written Examination (CBT)
  2. Physical Test
  3. Medical Test
2. Educational Qualification for Army ( Technical )
10th/+2
Intermediate Exam pass in Science with Physics, Chemistry, Maths and English with 50% marks in aggregate and 40% marks in each subject.
3. Age Limit for Army Technical
The Age limit for Army ( Technical ) are
  1. Minimum - 17.5 Years
  2. Maximum - 21 Years
4. Physical Measurement Test​ for Army Technical
In Army Technical physical test, the eligibility of candidates is measured as per height, chest and medical standards.
  • Height - 162
  • Weight - This is Based on BMI Scale
