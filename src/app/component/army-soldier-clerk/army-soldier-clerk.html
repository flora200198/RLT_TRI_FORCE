Army ( Soldier Clerk / Store Keeper Technical )
1. Selection Process for Army ( Soldier Clerk / Store Keeper Technical )
The Selection Process for Army ( Soldier Clerk / Store Keeper Technical ) are :-
  1. Written Examination (CBT)
  2. Physical Test
  3. Medical Test
2. Educational Qualification for Army ( Soldier Clerk / Store Keeper Technical )
  • Should have secured 50% marks In each subject and 60% marks in aggregate in class XII.
  • Should have studied English and Maths/Accounts / Book Keeping in Class XII or X and should have secured min 50% marks in each of these subjects, whether studied in Class XII or X.
  • Even if a candidate is a graduate or has a higher qualification percentage marks scored in class XII would be applicable towards his eligibility.
3. Age Limit for Army Soldier Clerk / Store Keeper Technical
The Age limit for Army ( Soldier Clerk / Store Keeper Technical ) are
  1. Minimum - 17.5 Years
  2. Maximum - 21 Years
4. Physical Measurement Test​ for Army Soldier Clerk / Store Keeper Technical
In Army Soldier Clerk / Store Keeper Technical physical test, the eligibility of candidates is measured as per height, chest and medical standards.
  • Height - 162
  • Weight - This is Based on BMI Scale
